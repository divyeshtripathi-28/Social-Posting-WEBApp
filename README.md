# Social Post App (Node.js, Express, MongoDB)

This project is a simple social posting web application built using Node.js, Express, MongoDB, and EJS.
Users can register, log in, create posts, edit posts, and like/unlike posts. Authentication is handled using JSON Web Tokens (JWT) stored in cookies, and passwords are securely hashed using bcrypt.

The project demonstrates user authentication, session management, database relationships, and CRUD operations in a full-stack web application.

# Features

# User Authentication
	•	User registration
	•	Secure password hashing
	•	Login and logout functionality
	•	JWT-based authentication stored in cookies

# Post Management
	•	Create new posts
	•	Edit existing posts
	•	View all posts on profile
	•	Like and unlike own posts

# Security
	•	Password hashing using bcrypt
	•	Authentication using JSON Web Tokens
	•	Protected routes using middleware
